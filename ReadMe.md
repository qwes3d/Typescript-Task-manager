# TypeScript Task Manager

A simple multi-page Task Manager application built with TypeScript, HTML, and CSS. The application allows users to create, view, edit, complete, and delete tasks while storing data in Local Storage.

---

# Features

- Add Tasks
- View Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Completed
- Multi-page Navigation
- Local Storage Persistence
- TypeScript Modular Architecture

---

# Technologies Used

- TypeScript
- HTML5
- CSS3
- Local Storage API

How to Run the Project
1. Install Dependencies
npm install
2. Compile TypeScript
npx tsc
3. Run Local Server
npx live-server pages
4. Open the Application
http://127.0.0.1:8080/pages/index.html
